# NFK Web Statistics

## PHP requirements

* PHP 5+ (works on 7)
* `apt install php-mbstring php-gd php-bz2 php7.0-xml`


## Schedule cron script to update graph images
`cron/update_graphs.sh`

## Write permissions
* `demos/`
* `mods/inc/dateflag.txt`

## Deny access from outside
* `mods`
* `langs`
* `inc`
* `cron`
